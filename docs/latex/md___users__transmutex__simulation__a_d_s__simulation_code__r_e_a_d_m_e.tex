This is the code repository for Transmutex. This code is used to simulate an Accelerator Driven System (ADS). This code is the property of Transmutex SA. All rights reserved.

\DoxyHorRuler{0}


In the root directory, you will find source files .cc and the header files .hh respectively in the following directories ./\+TMXElements/src/ and ./\+TMXElements/include/ These files constitutes general classes which are used to simulate specific features of a nuclear reactor (such as burn-\/up).

In the root directory, you will find source files .cc and the header files .hh respectively in the following directories ./\+G4-\/\+Neutron\+Transport/src/ and ./\+G4-\/\+Neutron\+Transport/include/ These files constitutes a GEANT4 simulation which is used to design the Transmutex\textquotesingle{}s ADS. The main file of this simulation is ./\+G4-\/\+Neutron\+Transport/ads\+\_\+transmutex.cc

The parameter file param.\+csv has to be configured by the user before each run. ~\newline
 This file defines the parameters of the simulation, it is located in {\ttfamily G4-\/\+Neutron\+Transport/} ~\newline
\hypertarget{md___users__transmutex__simulation__a_d_s__simulation_code__r_e_a_d_m_e_autotoc_md46}{}\doxysubsection{Description of the parameter file}\label{md___users__transmutex__simulation__a_d_s__simulation_code__r_e_a_d_m_e_autotoc_md46}
The parameter file can be visualized with MS Excel or a text file. ~\newline
 It consists in table organized as follows\+: ~\newline
 $\vert$\+Parameter name$\vert$\+Value$\vert$\+Units$\vert$\+Parameter description$\vert$\hypertarget{md___users__transmutex__simulation__a_d_s__simulation_code__r_e_a_d_m_e_autotoc_md47}{}\doxysubsection{Reading of the file}\label{md___users__transmutex__simulation__a_d_s__simulation_code__r_e_a_d_m_e_autotoc_md47}
The separators used to read the file are\+: ~\newline

\begin{DoxyItemize}
\item {\ttfamily ,} to separate each element ~\newline

\item {\ttfamily \textbackslash{}n},{\ttfamily \textbackslash{}r},{\ttfamily \textbackslash{}n\textbackslash{}r} or empty string {\ttfamily \char`\"{}\char`\"{}} to detect the end of a line ~\newline

\end{DoxyItemize}

One line corresponds to a single parameter except for the core configuration or a position vector (X,Y,Z) ~\newline
 To link a line to a specific parameter, the code reads the first element of each line which correspond to the parameter name. ~\newline
 The parameter name is hard-\/coded in the code {\ttfamily \mbox{\hyperlink{_t_m_x_input_parameters_8cc_source}{TMXInput\+Parameters.\+cc}}} that can be found in {\ttfamily ./\+TMXElements/src/}. ~\newline
